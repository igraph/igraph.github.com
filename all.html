Get started with R

Installation

igraph is on CRAN and can be installed from within R:

{% gist 6952578 %}

Requirements

On Linux (and other Unix-like systems, but not on OSX), a C and a C++ compiler is needed, and also the tool make. On Debian and Ubuntu Linux the build-essential package installs these.

Optional requirements

  • On Linux (and other Unix-like systems, but not on OSX), the libxml2 library, including header files, is needed for the ability of reading GraphML files. On Ubuntu and Debian Linux the lixml2 and libxml2-dev packages are needed.
  • The {% include rbasepkg p="tcltk" %} R package is needed for {% include rfun f="tkplot" %} and {% include rfun f="tkigraph" %} to work.
  • The {% include rpkg p="rgl" %} R package is needed for {% include rfun f="rglplot" %} to work.
  • The {% include rpkg p="ape" %} package is needed to plot nice dendrograms with {% include rfun f="dendPlot" %}.

Tutorials

  • Manual pages {% include rman p="igraph" %} and {% include rman p="graph.structure" %} are good places to start.
  • Some screenshots and examples are here.
  • Type in demo(package="igraph") (or use {% include rfun f="igraphdemo" %} if you have {% include rbasepkg p="tcltk" %}) to get a list of demos.
